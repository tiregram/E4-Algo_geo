## Code

Install the package need by the project

```bash
# for arch
pacman -S glew freeglut glm glfw
# add the other distrib ...

```

To run the demo program run this.

```bash
git clone http://github.com/tiregram/E4-Algo_geo
cd E4-Algo_geo

# generate make file
cmake .

# build
make

# run
./Mesh.bin
```
